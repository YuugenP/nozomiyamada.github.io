指示語（demonstrative）は世界の多くの言語に見られる語である。日本語はもちろん、英語・タイ語にも存在する。ちなみに「指示語」という品詞（part of speech）があるわけではなく、「何かを指示する機能を持った語」の総称である。具体的な品詞でいえば、代名詞（pronoun, สรรพนาม）や限定詞（determiner, ตัวกำหนด）などが相当する。this, many などのように、代名詞と限定詞のどちらでも使える語は多くある。

    代名詞の例
  • I, mine, yourself, it, that, there, another, everybody
  • ฉัน เธอ มัน ผู้ นี่ สิ่งใด ต่าง บรรดา
  • 私 先生 それ どちら 他 全て それぞれ
    限定詞の例
  • a, my, this, some, three, no, which, another, every, each
  • (ลักษณนาม)นี้ หลาย ห้า ทั้งหมด คนละ
  • どの そんな (…)の

日本語の指示語の研究については、佐久間 (1951) のものが有名である。日本語の「コ・ソ・ア」の分類は、従来は単に距離が近いか中程度か遠いかというだけのものであったが（距離区分説）、佐久間は「話し手・聞き手の近くにあるかどうか」という点に注目した（人称区分説）。

  • コ：話し手の近くにあるもの e.g. この本面白いよ
  • ソ：聞き手の近くにあるもの e.g. その本面白い？
  • ア：どちらからも離れているもの e.g. あの本面白そう

ただ、これらの指示語の使い方は、存在するものを直接指示する現場指示 (spatial deixes) である。指示する内容が文脈に依存するかという文脈指示 (discourse deixes) の機能についてより深く研究したものには、久野 (1973) がある。久野は「聞き手の知識」に基づいて、「ソ・ア」を以下のように分類した。（以下の例の * は ungrammatical を意味する）

  • ソ：話し手は知っているが、聞き手は知らない e.g. 昨日本屋に行ったら、そこ/*あそこで先生に会いました
  • ア：話し手、聞き手共に知っている e.g. 昨日一緒に行った*その/あのレストランの名前は何だっけ？

ただし、この分類はあくまで「聞き手の知識」を前提としているため、独り言などのように聞き手がいない場合には適用できない。そこで 田窪・金水 (1996) は認知科学 (cognitive science) の考えを適用し、文脈指示を以下のように分類した。

  • ソ：伝聞や推論など、一時的（間接的）記憶である「Indirect Domain」にあるものを指示する
  • ア：直接経験し、長期記憶となった「Direct Domain」にあるものを指示する

これはいわば、コンピュータの RAM と HDD のようなものである。具体的な例を見てみると、

  1. 今度新しい先生が来るってさ - その/*あの先生、男性？
  2. 「こころ」という小説知ってる？ - ああ、それ/*あれ知ってるよ。その/*あの本面白い？
  3. 「こころ」という小説知ってる？ - ああ、それ/*あれ知ってるよ。その/あの本面白いよね

1と2の例で「ア」が使えないのは、今話している内容が「一時記憶領域」にあるものだからである。特に2の後半の「その本」の部分は、自分が知ってはいても読んだことがなく「直接経験」ではないため、「あの」を使うことはできない。一方で3の例では、まず「こころ」という小説を話題にする＝一時記憶領域に置くことで最初は「それ」を使うが、自分が読んだ経験があるため、二回目は長期記憶にアクセスすることで「あの本」という言い方をすることも可能である。この「直接経験であるかどうか」ということは、独り言での回想などで「ア」を使う（e.g. あの時は…）ことの説明にもなる。

このように指示語の使い方とその説明には様々なものがあり、ここで挙げたもの以外にもまだ多くが存在する。それぞれの説明の差はあれど、指示語の性質を知るために共通して行っているのは replacement test (代用テスト) である。つまり、「それ/あれ」のように一単語のみを置き換えることによって ungrammatical となるかどうかをテストするものである。これは Leonard Bloomfield による Immediate Constituent Analysis (IC分析) をルーツとし、チョムスキー以後に発展した手法である。grammatical であるかどうかの判断にはネイティヴスピーカーが必要ではあるが、自身の仮説を検証する方法論として言語学では非常に一般的かつ重要なものである。今後研究する際にもぜひ留意して欲しい。
